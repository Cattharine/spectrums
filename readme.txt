Использование программы
Для использования программы необходимо занести строки, соответствующие исследуемым откликам, в файл in.txt.
Приведем формат строки файла in.txt:
{имя отклика}{символ(-ы) отступа}{строчка, соответствующая отклику}
где {имя отклика} – может содержать любые символы кроме символов отступа, используется в файле вывода out.txt для идентификации отклика,
{символ(-ы) отступа} – некоторое количество пробельных символов или символов табуляции,
{строчка, соответствующая отклику} – набор нулей и единиц, соответствующий значениям отклика при разных уровнях факторов, упорядоченных в лексикографическом порядке. Понятно, что длина такой строки должна быть степенью 2.
Пример такой строки:
xor	0110
Замечание. Если строка входных данных не соответствует формату, она не будет обработана.
Далее производится запуск программы.
Замечание. Скорость поиска спектра при малых n отличается не значительно, но при больших n (n > 10) программа может работать достаточно длительное время.
Результат будет записан в файл выходных данных out.txt в следующем формате:
{спектр} = [имена обработанных откликов]
Пример выходных данных (соответствующий примеру входных данных):
0, 1, 1 = [xor]
